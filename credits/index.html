"Dexter Learn" is a non-real machine learning app made by Matthew Nerger, a front end web developer and writer. It has no actual functionality (as of now), and works something like a do-nothing machine, except for the web. Essentially, it imitates an advanced machine learning application but does absolutely nothing but repeat the words inputed in it. If anything, it's a way for me to experiment with different visual styles and make bad jokes, two of my favorite hobbies.
